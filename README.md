# Next.js User Dashboard

A Next.js application that displays a list of users fetched from a public API using Server-Side Rendering (SSR).

## Features

- **Server-Side Rendering (SSR)** via `getServerSideProps()`
- Fetches users from [JSONPlaceholder API](https://jsonplaceholder.typicode.com/users)
- Displays Name, Email, Company, and City per user
- **Real-time search** by user name
- Responsive grid layout

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Tech Stack

- Next.js 13
- React 18
- CSS Modules
